/api/installations get

get

Retourne la liste de toutes les installations

get /api/installations

Retourne la liste de toutes les installations

  • Request
  • Response

Query Parameters

  • arrondissement: (string)

    Si ajouté, retourne la liste de toutes les installations selon cet arrondissement

    Example:

    /api/installations?arrondissement=Ahuntsic

HTTP status code 200

Retourne une liste d'installations

Body

Media type: application/json

Type: object

Properties
  • glissades: required(array of Glissade)

    Items: Glissade

    • nom: required(string)
    • arrondissement: required(object)
      • nom_arr: required(string)
      • cle: required(string)
      • date_maj: required(datetime)
    • ouvert: required(boolean)
    • deblaye: required(boolean)
    • condition: required(string - default: N/A)
  • patinoires: required(array of Patinoire)

    Items: Patinoire

    • nom_arr: required(string)
    • nom_pat: required(string)
  • piscines: required(array of Piscine)

    Items: Piscine

    • id_uev: required(integer)
    • type: required(string)
    • nom: required(string)
    • arrondisse: required(string)
    • adresse: required(string)
    • propriete: required(string)
    • gestion: required(string)
    • point_x: required(string)
    • point_y: required(string)
    • equipeme: required(string)
    • longitude: required(number)
    • latitude: required(number)

Example:

{
    "glissades": [
        {
        "nom": "Aire de glissade ,Don-Bosco",
        "arrondissement": {
        "nom_arr": "Rivière-des-Prairies - Pointe-aux-Trembles",
        "cle": "rdp",
        "date_maj": "2021-10-18 13:45:13"
        },
        "ouvert": 0,
        "deblaye": 0,
        "condition": "N/A"
        }
    ],
    "patinoires": [
        {
        "nom_arr": "Ahuntsic - Cartierville",
        "nom_pat": "Aire de glissade, Ahuntsic ()"
        },
        {
        "nom_arr": "Ahuntsic - Cartierville",
        "nom_pat": "Aire de glissade, Des Hirondelles ()"
        }
    ],
    "piscines": []
}
